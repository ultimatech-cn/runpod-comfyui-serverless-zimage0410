# Search Ranking

模型候选链接默认按以下优先级排序：

1. 文件名完全匹配
2. 文件名 stem 完全匹配
3. stem 高相似度匹配
4. 仓库名或版本名包含主要关键词
5. 候选结果与模型类型语义匹配

默认搜索源：

- Hugging Face
- Civitai

每个模型最多保留：

- 1 个推荐候选
- 2 个备选候选

置信度规则：

- `high`：文件名完全匹配或接近完全匹配
- `medium`：关键词与类型匹配较强，但仍需人工确认
- `low`：只有模糊关键词匹配

重要原则：

- 搜索结果默认不是最终真值
- 所有搜索补出的链接都应标记 `verification_required`
